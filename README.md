# tangdao

#### 介绍
综合管理系统，采用 springboot2，spring-security，spring-session，mybatis-plus，redis，fst，beetl 等框架，已完成基础权限的管理，封装了内置组件，布局模板等有利于快速开发的基础项目。基于系统可以开发，后台权限管理系统，OA管理系统，短信管理系统，CRM管理系统，企业管理系统，电商后台管理等。
