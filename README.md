# Tangdao

#### 介绍
Tangdao 是基于角色的权限管理系统（RBAC），采用Springboot开发。系统简单易懂，前端使用Vue、Quasarframework开发，页面简洁美观。后端核心框架使用Springboot、Mybatis-plus、SpringSecurity为主要，扩展基于框架的权限校验、参数校验、统一异常、统一响应的通用功能。
